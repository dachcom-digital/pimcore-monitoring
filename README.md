# Pimcore Monitoring

#### Requirements
* Pimcore 5.

## Installation

```json
"require": {
    "dachcom-digital/monitoring" : "~2.0.0",
}
```

## Fetch Data
```php
GET /plugin/Monitoring/watch-dog/fetch?secret=yourAuthKey
```
